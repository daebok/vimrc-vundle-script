# vimrc-vundle-script
여러 리눅스 환경에서의 vimrc, vim plugin 파일 동기화를 위한 저장소

## Vundle 설치
<pre>
$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
</pre>

## .vimrc의 플러그인 설치
<pre>
$ vim
: VundleInstall
</pre>

## TagList 사용을 위한 ctags 설치
### Debian
<pre>
$ sudo apt-get install ctags
</pre>
### Redhat
<pre>
$ sudo yum install ctags
</pre>
